# g_CiEnabled-Hunter

## Description

This tool is designed to specify and locate the address of `g_CiEnabled`, which is a critical kernel variable responsible for enabling and disabling Driver Signature Enforcement (DSE) in Windows systems.

## Purpose

`g_CiEnabled` is a kernel-mode variable that controls whether the system enforces driver signature verification. This hunter tool helps identify its memory location for research and security analysis purposes.

## Features

- Locates the memory address of `g_CiEnabled` kernel variable
- Assists in understanding DSE (Driver Signature Enforcement) mechanisms
- Useful for security researchers and system-level developers
